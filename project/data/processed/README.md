This folder contains datasets generated by the project's *cleaning and merging pipeline*:

> *Note:* Processed datasets are *not uploaded* to GitHub. They are generated automatically when running the pipeline.

*Storage & Usage:*  
- Generated as CSV or Parquet files in data/processed/.
- Automatically read by downstream pipelines: feature engineering, modeling, and deployment.
